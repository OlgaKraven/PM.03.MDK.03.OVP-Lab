# PM.03.MDK.03.OVP-Lab

Учебный репозиторий с лабораторными работами помеждисциплинарному курсу **МДК.03 «Оптимизация веб-приложений»** (ОВП) для студентов СПО по специальности 09.02.07 «Информационные системы и программирование» и родственных направлений.

Репозиторий используется для:
- отработки базовых навыков **вёрстки** (HTML, CSS);
- знакомства с **клиентским JavaScript**;
- освоения принципов организации файлов веб-проекта;
- практики работы с **Git / GitHub** и оформления отчётности по лабораторным.
